## 🛠️ Installation & Usage

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/calculator-app.git
   cd calculator-app
   ```

2. **Install dependencies** (if applicable)

   ```bash
   npm install
   ```

3. **Run the app locally**

   ```bash
   npm start
   ```

4. **Open in your browser**

   ```
   http://localhost:3000
   ```
